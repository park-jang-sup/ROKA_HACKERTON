# 스마트 군수물자 재물조사 시스템 (ROKA Hackathon)

사진 한 장으로 총기류를 자동 인식하고, 편제 정수 대비 과부족을 즉시 판정하는 AI 기반 군수물자 재물조사 모바일 앱입니다.

## 프로젝트 개요
* **플랫폼:** Android / iOS (Flutter)
* **개발 기간:** 2026.06 ~ 2026.07
* **개발 인원:** 3인 (팀 프로젝트)
* **팀 프로젝트 원본 저장소:** [https://github.com/ThubanA1pha/ROKA_HACKERTON](https://github.com/ThubanA1pha/ROKA_HACKERTON)

## 주요 기능
* **촬영 기반 자동 재고 등록:** YOLOv8(TFLite) 모델을 통해 K1, K2, K2C1 기종 및 수량 즉시 인식.
* **과부족 자동 판정:** 실시간 인식 수량과 기종별 편제 정수를 즉각 비교하여 상태 자동 라벨링.
* **실시간 데이터 동기화:** Firebase Firestore 스트림을 통한 여러 단말 간 데이터 즉시 반영.
* **총기 일련번호 관리:** 서버 파이프라인(FastAPI/EasyOCR)을 이용한 정밀 일련번호 인식 지원.

## 기술적 주안점 및 문제 해결

* **온디바이스 AI 전환 및 최적화:** 현장의 네트워크 불안정성과 데이터 보안을 위해 서버 기반 추론에서 TFLite 기반 온디바이스 추론으로 설계를 변경했습니다. 이를 위해 YOLOv8 모델을 FP16으로 경량화하여 모바일 환경에서 실시간 추론이 가능하도록 최적화했습니다.
  
* **Firebase를 활용한 안정적인 데이터 동기화:** 재물조사 시 실시간으로 변하는 데이터를 관리하기 위해 Firestore의 보안 규칙을 정교하게 설정했습니다. 인증된 사용자만 접근 가능하도록 데이터 접근 권한을 제어하여 군 보안 요구사항을 준수했습니다.

* **하이브리드 데이터 파이프라인 구축:** 온디바이스 추론(신속성)과 서버 추론(정밀성, FastAPI/EasyOCR)의 역할을 명확히 분리하여, 상황에 맞는 최적의 추론 경로를 설계하고 연산 효율을 높였습니다.

## 시스템 구성

* flutter_ui/          # Flutter 모바일 앱 — UI, TFLite 추론
* src/                 # YOLO 추론 파이프라인, FastAPI 서버
* db/                  # Firebase 스키마 및 보안 규칙
* data/, models/       # 학습 데이터셋 및 가중치 파일
